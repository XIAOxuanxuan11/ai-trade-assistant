# AI外贸文档助手 (ai-trade-assistant)

## 项目简介
专为精密工业OEM/ODM出口场景设计的AI文档处理工具，可自动翻译英文技术规格书、整理客户邮件要点、生成专业报价函，大幅提升外贸团队的工作效率。

## 核心功能
- **技术文档翻译**：将英文铸锻、冲压等工业技术文档，精准翻译为中文要点摘要，保留关键参数与标准。
- **客户邮件整理**：自动提取英文客户邮件中的需求、问题和跟进要点，生成结构化的中文跟进计划。
- **报价函生成**：根据中文报价单，自动生成符合国际商务规范的英文专业报价函。

## 使用示例
输入：一份英文技术文档片段
输出：中文要点摘要

## 技术栈
- **大模型**：ERNIE Bot / ChatGPT
- **开发语言**：Python
- **文档处理**：PyPDF2, python-docx
- **部署方式**：本地运行，支持批量处理
